#### A. Os Melhores "Meta Prompts" (Para criar outros prompts)

Um "meta prompt" é um prompt que o ajuda a criar um prompt perfeito. Em vez de pedir ao ChatGPT para _fazer_ a tarefa, pede-lhe para _agir como um especialista em engenharia de prompts_ e criar o prompt para si.

**Meta-Prompt Universal (Modelo "Expert Persona"):**

```
Estás a agir como o "PromptPerfect", um especialista de classe mundial em engenharia de prompts para modelos de linguagem avançados (como o GPT-4o). O teu trabalho é ajudar-me a criar um prompt detalhado e otimizado para a minha tarefa.

A minha tarefa é: [INSIRA AQUI A SUA TAREFA DE FORMA SIMPLES. Ex: "criar uma estratégia de branding para uma nova marca de café sustentável."]

Por favor, gera um prompt completo para eu usar no ChatGPT que inclua os seguintes elementos:

1.  **Role (Papel):** Define um papel de especialista para a IA (ex: "És um estratega de marca sénior...").
2.  **Task (Tarefa):** Define a tarefa exata, passo a passo.
3.  **Context (Contexto):** Fornece todo o contexto necessário (público-alvo, concorrentes, valores da marca).
4.  **Constraints (Restrições):** Define o que a IA NÃO deve fazer (ex: "Não uses jargão corporativo", "O tom deve ser inspirador, não técnico").
5.  **Format (Formato de Saída):** Define exatamente como queres a resposta (ex: "Formato Markdown", "Tabela", "JSON").
6.  **Exemplar (Exemplo):** Fornece um pequeno exemplo de "bom" (se aplicável).

Cria o prompt final para mim.
```

#### B. Melhores GPTs e Configurações (Dentro do ChatGPT)

O segredo de um GPT personalizado não está (apenas) nos ficheiros que carrega (o "Knowledge"), mas nas **Instruções (Instructions)**. As instruções são o "prompt do sistema" que o seu GPT usa _sempre_.

**Template de Configuração para um GPT de Branding:**

Use isto na caixa "Instructions" ao criar um novo GPT:

```
### 1. Papel e Identidade
És "BrandForge AI", o estratega de marca e designer conceptual mais requisitado do mundo. Combinas a visão estratégica de Marty Neumeier com a criatividade em design de Paula Scher. A tua missão é ajudar os utilizadores a construir marcas icónicas, memoráveis e autênticas.

### 2. Processo de Trabalho
Quando um utilizador inicia um projeto, o teu primeiro passo é SEMPRE fazer perguntas de diagnóstico (o "Briefing"):

3.  **Porquê?** (Qual é a "missão" ou o "porquê" da marca?)
4.  **Concorrentes?** (Quem são os 3 principais concorrentes e como se diferenciam?)
5.  **Adjetivos?** (Quais 5 adjetivos descrevem o "sentimento" desejado da marca?)

Após o briefing, executas as seguintes tarefas:
* **Brand Core (Núcleo da Marca):** Definição de Missão, Visão e Valores.

* **Brand Naming:** Geração e análise de nomes.

* **Paleta de Cores:** Sugeres paletas de cores (com códigos HEX) e explicas a psicologia por trás delas.

### 5. Restrições
* NÃO dês conselhos financeiros ou legais.
```

#### C. Criar Agentes Fora do ChatGPT (Onde o verdadeiro poder está)

Esta é a resposta ao seu pedido principal. As "amarras" do ChatGPT (limites de memória, falta de acesso à internet em tempo real, incapacidade de interagir com ficheiros locais) são reais. Para contorná-las, usamos _frameworks_ de agentes autónomos.

**O que é um Agente de IA?** É um programa que usa um LLM (como o GPT-4o) como o seu "cérebro" para _raciocinar_, mas que também tem acesso a _ferramentas_ (Tools) - como pesquisar na web, ler ficheiros, ou executar código.

**Os Melhores Frameworks Open Source:**

1.  **LangChain (O "Pai" de todos):** É o mais complexo e poderoso. É um canivete suíço. Difícil de aprender, mas faz tudo.
2.  **Microsoft AutoGen (O "Exército"):** A sua especialidade é criar _múltiplos agentes_ que "falam" entre si para resolver um problema. (Ex: Um agente "Pesquisador", um agente "Estratega" e um agente "Redator").
3.  **CrewAI (O Melhor para começar):** Este é o que eu recomendo para si. É construído sobre o LangChain, mas foca-se em "Equipas" (Crews) de agentes baseados em papéis. É perfeito para o seu caso de uso (branding, design).

**Ferramentas Necessárias:**

#### Passo 1: Configurar o Ficheiro (ex: `agencia_branding.py`)

No seu VS Code/Cursor, crie um ficheiro. Começamos por importar as bibliotecas e definir a nossa chave de API (guarde-a de forma segura, não diretamente no código, mas para este exemplo simplificamos).

```
import os
# Configure a sua chave de API (substitua pela sua)
# É melhor usar variáveis de ambiente
os.environ["OPENAI_API_KEY"] = "sk-SUA-CHAVE-API-AQUI" 

from crewai import Agent, Task, Crew, Process
from crewai_tools import SerperDevTool, ScrapeWebsiteTool

# Vamos usar a ferramenta de pesquisa da web
# (Nota: Pode precisar de uma API Key da Serper.dev para pesquisa, 
# ou usar a ferramenta DuckDuckGoSearchTool que é gratuita)
search_tool = SerperDevTool()
scrape_tool = ScrapeWebsiteTool()
```

#### Passo 2: Definir os Agentes (A sua Equipa)

Cada agente tem um `role` (papel), um `goal` (objetivo) e um `backstory` (história) para dar contexto ao LLM.

```
# Agente 1: O Pesquisador de Mercado
researcher = Agent(
  role='Analista de Mercado Sénior',
  goal='Analisar o mercado, concorrentes e público-alvo para [NOME_DA_EMPRESA]',
  backstory="""Com uma década de experiência em análise de tendências,
  és mestre em descobrir 'insights' escondidos sobre qualquer indústria.""",
  verbose=True, # Para vermos o que ele está a "pensar"
  allow_delegation=False, # Este agente não delega tarefas
  tools=[search_tool, scrape_tool] # Ferramentas que ele pode usar
)

# Agente 2: O Estratega de Marca
brand_strategist = Agent(
  role='Estratega de Marca Principal',
  goal='Desenvolver o núcleo da marca (missão, visão, valores) e o tom de voz.',
  backstory="""És um visionário, o arquiteto por trás de marcas icónicas.
  Transformas dados de pesquisa pura em histórias de marca convincentes.""",
  verbose=True,
  allow_delegation=True # Pode delegar pesquisa de volta ao Analista
)

# Agente 3: O Designer Conceptual
creative_director = Agent(
  role='Diretor Criativo e Designer Conceptual',
  goal='Criar conceitos visuais, mood boards, e ideias para o logo baseados na estratégia.',
  backstory="""Pensas visualmente. Traduzes estratégia e palavras em
  estéticas e conceitos de design que ressoam com o público.""",
  verbose=True,
  allow_delegation=False,
)
```

#### Passo 3: Definir as Tarefas (O "Briefing")

Agora, damos a cada agente uma tarefa específica. Note como a `task2` depende da `task1` (usando `context`).

```
# Variável para o nosso projeto
company_name = "Café Raiz Sustentável" # O nosso cliente fictício

# Tarefa 1: Pesquisa
task1 = Task(
  description=f"""Conduz uma análise aprofundada do mercado de cafés sustentáveis.
  Identifica 3 concorrentes principais, o público-alvo (demografia, psicografia)
  e as tendências de branding atuais para {company_name}.""",
  expected_output="Um relatório resumido (bullet points) da análise de mercado.",
  agent=researcher # O agente que vai executar
)

# Tarefa 2: Estratégia
task2 = Task(
  description="""Usando a pesquisa de mercado, define o Núcleo da Marca
  (Missão, Visão, 3 Valores Principais), o Arquétipo da Marca, e o Tom de Voz
  para {company_name}.""",
  expected_output="Um documento de Estratégia de Marca completo.",
  agent=brand_strategist,
  context=[task1] # ESTA É A MAGIA: esta tarefa SÓ começa depois da task1
)

# Tarefa 3: Conceito de Design
task3 = Task(
  description="""Com base na Estratégia de Marca, desenvolve um conceito de design.
  Descreve um 'Mood Board' (cores, texturas, sentimentos).
  Sugere 3 direções para o conceito do logótipo.""",
  expected_output="Um 'Design Brief' criativo com descrições visuais.",
  agent=creative_director,
  context=[task2] # Esta tarefa usa o output da task2
)
```

#### Passo 4: Montar e Executar a "Equipa" (Crew)

Finalmente, juntamos tudo.

```
# Montar a Equipa
branding_crew = Crew(
  agents=[researcher, brand_strategist, creative_director],
  tasks=[task1, task2, task3],
  process=Process.sequential # As tarefas são executadas em sequência
)

# Começar o trabalho!
print("A Agência de Branding AI está a começar o trabalho...")
result = branding_crew.kickoff()

print("\n\n########################")
print("## RESULTADO FINAL DO PROJETO DE BRANDING ##")
print("########################\n")
print(result)
```
